Good Shepherd Parish Youth Council — Finance System

A full-featured financial management web app built with React.

## Features
- 📊 Dashboard with charts and financial overview
- 💰 Income & Expense tracking
- 📈 IGF (Internally Generated Funds) tracker
- 📋 Full transaction ledger with filters
- 📑 Reports with CSV export
- 👥 Multi-role user management

## Local Development

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy to GitHub Pages

```bash
# 1. Update package.json homepage field to your GitHub Pages URL:
#    "homepage": "https://YOUR_USERNAME.github.io/good-shepherd-finance"

# 2. Install and deploy
npm install
npm run deploy
```
